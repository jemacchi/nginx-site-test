Simple web page for testing a Git rollout action on determined deployments

This page is shown as part of a PoC on which a git-rollout-operator looks for changes in a git repo, and then rollouts a defined deployment.

In our case the deployment is a nginx which reloads the content of the exposed html with an initcontainer which clones the repo content on the publich html folder.

When a git commit is detected, the operator triggers a rollout which in turns executes the initContainer command.

We set the webpage to reload every 5 secs so, for our PoC we can see how the pages automatically updates the content.
